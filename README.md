                 PERSONAL PRODUCTIVITY AND LIEF MANAGEMENT (Task Overflow)

   Project Description -

This project is a simple web-based application developed using HTML, CSS, and Vanilla JavaScript.
The application helps users manage daily tasks, organize responsibilities, and improve personal productivity.
All functionality is implemented using JavaScript and DOM manipulation without using any frontend frameworks.

   Problem Statement -

Managing multiple daily tasks can be challenging without a proper system.
People often forget tasks or lose track of completed and pending work.

This project provides a simple digital task management solution where users can add, track, complete, and remove tasks, helping them stay organized and productive.

   Features Implemented -

1.Name-based start using a “Get Started” screen
2.Greeting message displayed as Hi <name> on the planner page
3.“How It Works” guide explaining how to use the application
4.Add tasks with priority levels (High, Medium, Low)
5.Mark tasks as completed or pending
6.Delete tasks from the task list
7.Productivity summary showing total, completed, and pending tasks
8.Logout functionality that clears all tasks and resets the application

   DOM Concepts Used -

1.Dynamic creation of elements using document.createElement()
2.Updating content using innerText
3.Adding and removing CSS classes using classList
4.Event handling for button clicks and form submission
5.Event delegation for handling task actions
6.Conditional DOM updates based on task status

   Steps to Run the Project -

1.Download or clone the project repository from GitHub.
2.Open the downloaded project folder on your system.
3.Ensure the folder contains index.html, style.css, and script.js.
4.Open the index.html file in any modern web browser such as Chrome, Edge, or Firefox.
5.Enter your name, click Get Started, and begin using the planner

   Known Limitations -

1.Tasks are cleared when the user logs out
2.Tasks are not saved across sessions
3.No task priority or due date feature
4tas.Designed for single-user usage only
